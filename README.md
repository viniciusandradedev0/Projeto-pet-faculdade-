# 🐾 PAWS PLACE

> Site dedicado à adoção responsável de animais que sofreram com o abandono.

## 📖 Sobre o Projeto

O **PAWS PLACE** é um projeto em HTML, CSS e JavaScript que conecta pessoas a animais resgatados, oferecendo uma vitrine simples e acolhedora para gatos e cachorros que aguardam por um novo lar.

## ✨ Funcionalidades

- 📋 Listagem dinâmica de animais (gatos e cachorros)
- 🖼️ Cards com foto, nome, idade e link para detalhes
- 📱 Layout responsivo (mobile-first)
- ♿ Estrutura semântica e acessível

## 🛠️ Tecnologias

- **HTML5** semântico
- **CSS3** com design tokens e variáveis customizadas
- **JavaScript (ES6+)** com módulos nativos
- **JSON** como base de dados local
